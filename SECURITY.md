# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Dave, please report it responsibly.

**Email:** drorlazar@sett.ai

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Assessment:** Within 7 days
- **Fix:** Within 30 days (for confirmed vulnerabilities)

## Scope

This policy applies to the Dave repository code only. Third-party dependencies (Three.js, Font Awesome) should be reported to their respective maintainers.

## Supported Versions

Only the latest version on the `main` branch is actively supported.
