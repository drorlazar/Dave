# Dave FTUE - First Time User Experience Tour

**Date**: 2026-02-15 | **Branch**: `feature/ftue`

## Context

First-time users land on an empty DAVE app with no guidance. The FTUE gives Dave himself the role of tour guide — he physically flies to each UI element, spotlights it, and explains it in his signature voice. The tour is ~80 seconds, skippable at any time, and ends with Dave turning himself off and telling the user how to re-enable him.

## First-Visit Detection

Auto-trigger when ALL conditions are true:
1. `localStorage.getItem('dave_ftue_completed')` is NOT `'true'`
2. `localStorage.getItem('dave_ftue_skipped')` is NOT `'true'`
3. `localStorage.getItem('dave_fullmode_first')` is `null` (never used Dave before)

Storage keys:
- `dave_ftue_completed` — set `'true'` on natural tour completion
- `dave_ftue_skipped` — set `'true'` on user skip
- `dave_ftue_version` — `'1'` (allows re-triggering on major updates)

### Boot Sequence (~2s)
Matrix-typewriter text flashes in viewer center:
```
> FTUE MODULE LOADED
> PERSONALITY: ENTHUSIASTIC (OVERRIDE)
> BEGINNING GUIDED TOUR...
```

**Position**: Default (bottom-right) | **Emotion**: PROUD | **Effect**: jitter
```
Oh. OH. A new face. *adjusts pixels* Welcome to D.A.V.E. — that's me.
I'm your asset viewer. I also have opinions. Let me show you around MY place.
```

### Step 1: Source Picker (`#sourceDropdown`, ~7s)
**Emotion**: WARM
```
This is where the magic starts. Pick a local folder, connect to S3, or Google Drive.

**Emotion**: CURIOUS

Search bar. Finds files, filters results. You can also paste cloud URLs

### Step 3: Viewer Container (`#viewerContainer`, ~6s)
**Emotion**: EXISTENTIAL
```
This is the main stage. Your files show up here as tiles.
Click any tile to preview it full-screen. Right now it's... empty.
*stares into void* I'm used to it.
```

### Step 4: Size Slider (`.size-control`, ~6s)
**Emotion**: AMUSED
```
Tile size control. Slide right for BIG. Slide left for tiny.
I don't judge your preferences.

### Step 5: Sort & Filter (`#sortBtn` + `#assetTypeFilterToggleBtn`, ~7s)
**Emotion**: SMUG
```
Sort by name, size, type, or date. The eye icon next to it? That's your

### Step 6: Settings Gear (`#settingsDropdown`, ~7s)
**Emotion**: SASSY | **Special**: Leave a post-it on the gear icon

Settings. Themes — I look good in all fourteen of them. Cloud storage connections.

### Step 7: Folder Tree (`#treeFolderToggle`, ~6s)
**Emotion**: NEUTRAL
```

### Step 8: Page Controls (`.pageControls`, ~5s)
**Emotion**: NEUTRAL

### Farewell (3-part, ~12s)
Dave returns to home position.

**Part 1** — WARM:
```
And that's the tour. Not bad for a web app, right?
```

**Part 2** — SAD + single tear (2s later):
```
Now here's the thing. I'm going to turn myself off.
You can find me in Settings — that gear icon — under "Full Dave Mode".
Toggle me ON whenever you want the company. I'll be... waiting. In the void.
```

**Part 3** — PROUD + fireworks (2s later):
```
*straightens pixels* It was nice meeting you. Now go view some assets.
That IS what we're both here for. ...Right?
```

Then: `DaveMode.disable(false)`, set `dave_ftue_completed = 'true'`.

## Skip Mechanism

- **Skip button**: Bottom-left corner, subtle text: `Skip Tour [ESC]`
- **ESC key**: Also triggers skip
- **On skip**: Clear highlights, Dave flies home, shows:

```
No worries. Quick version: drop files here, they show up as tiles,
click to preview. I'm under Settings > Full Dave Mode if you want me around.
*salutes*
```

Emotion: WARM. Then disable Dave after bubble dismisses. Set `dave_ftue_skipped = 'true'`.

## Special Features

### Spotlight Overlay
Semi-transparent dark overlay (`rgba(0,0,0,0.6)`) with CSS `clip-path: polygon(...)` cutout around target element + green glow border. Cutout recalculates on window resize.

### Trail Breadcrumbs
Dave leaves green trail chars while flying between stops. Each visited stop gets a persistent `+` waypoint marker that fades out during farewell.

### Progress Dots
Row of small green dots at bottom-center. Current dot pulses, completed dots glow solid, future dots are dim. Gives users sense of tour length.

### Post-It Note
After Step 6 (Settings), Dave leaves a CRT-styled post-it on the gear icon: `"Full Dave Mode" is in here - D.` Persists for 30 seconds after tour ends.

### Emotional Arc
PROUD → WARM → CURIOUS → EXISTENTIAL → AMUSED → SMUG → SASSY → NEUTRAL → WARM → SAD → PROUD (narrative arc, not a checklist)

## Technical Architecture

### Class: `_DaveFTUE` (singleton export as `DaveFTUE`)

**State machine**: IDLE → STARTING → STEP_0..N → FAREWELL → DONE (with SKIPPED branch)

**Key methods**:
- `init()` — Check localStorage, auto-start if first visit (2s delay)
- `start(force = false)` — Entry point. `force` bypasses localStorage (for debug)
- `_prepareTour()` — Enable Dave Mode (silent), suppress DAV-9000, suspend idle, stop cursor follow
- `_runStep(index)` — Fly to target, spotlight, speak, wait, advance
- `_createOverlay()` / `_updateSpotlight(rect, padding)` — Dimming + clip-path
- `_createProgressDots()` / `_updateProgress(step)` — Bottom dots
- `_showBootSequence()` — Matrix text intro
- `_leavePostIt()` — Post-it on settings gear
- `_farewell()` / `_skipFarewell()` — Goodbye sequences
- `skip()` — User-initiated skip
- `_cleanup()` — Remove all tour DOM, restore state
- `_wait(ms)` — Abortable delay (checks `_aborted` flag)

**Pacing**: Hybrid — auto-advance after ~6-8s OR user clicks bubble dismiss (x) to advance immediately.

**Integration**:
- `DaveMode.enable(false)` at start (no reaction message)
- `cancelTakeover()` + `destroyDAV9000Terminal()` to suppress terminal
- `DaveAlive._ftueActive = true` to suppress idle behaviors
- `DaveMode._stopCursorFollow()` during flight
- `DaveMode._showBubble(text, { force: true })` for speech

## Edge Cases

- **Element not found**: Skip step silently, move to next
- **Window resize**: Debounced recalculation of spotlight clip-path
- **Files loaded during tour**: Tour continues (Step 3 "empty" joke may be slightly off — acceptable)
- **Dave already enabled**: Skip the `enable()` call, still run tour
- **Double trigger**: Guard `_running` flag
- **Reload mid-tour**: Re-triggers next load (localStorage not set yet — acceptable)

2. **Full playthrough**: All 9 steps + farewell play correctly, Dave flies/highlights/speaks
3. **ESC skip**: Press ESC mid-tour → highlights clear → skip farewell → Dave disables
4. **Skip button**: Click "Skip Tour" → same as ESC

7. **Returning user**: Set `dave_fullmode_first` in localStorage → FTUE does NOT trigger

9. **Missing element**: Hide `#treeFolderToggle` via DevTools → that step skipped without error
10. **Debug panel**: All 7 sections render, all existing buttons still work, FTUE button in correct section
